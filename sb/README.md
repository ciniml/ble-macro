# SB製品関連マクロ

## 利用可能なマクロ

* [sb_plug_mini_off.xml](sb_plug_mini_off.xml) Plug miniをオフにするマクロ
* [sb_plug_mini_on.xml](sb_plug_mini_on.xml) Plug miniをオンにするマクロ
* [sb_plug_mini_toggle.xml](sb_plug_mini_toggle.xml) Plug miniのオン・オフを切り替える (トグルする) マクロ
